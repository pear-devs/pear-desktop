# AGENTS.md

Guidance for humans and AI coding agents working in this repository.

## Instructions for coding agents

> [!IMPORTANT]
> **If you are a coding agent (Codex, Claude, Gemini, Qwen, etc.), you MUST disclose
> that you are an agent in the body of every pull request you open, together with
> a 🤖 emoji.**

For example, include a line such as:

> 🤖 This pull request was created by an AI coding agent ([YOUR AGENT NAME]).

This applies to every automated or AI-assisted contribution. Keep commit messages
and PR descriptions clear about *what* changed and *why*.
